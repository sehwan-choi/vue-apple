  - extension 설정
    1. material icon
       mac :
         상단 code -> 기본 설정 -> 파일 아이콘 테마 -> material icon 설정
       window :
         상단 file -> preference -> theme -> file icon theme -> material icon 설정

    2. night owl
       mac :
         상단 code -> 기본 설정 -> 색테마 -> night owl 설정
       window :
         상단 file -> preference -> theme -> color theme -> night owl 설정
